Дана матрица смежности 5х5

0  1  1  0  0	-из 0 путь в 1 и 2
1  0  0  1  0	-из 1 путь в 0 и 3
1  0  0  1  0	-из 2 путь в 0 и 3
0  1  1  0  1	-из 3 путь в 1, 2 и 4
0  0  1  1  0	-из 4 путь в 2 и 3

Необходимо найти такое начало, из которого мы начнём и не вернёмся по поду дальнейшего движения. 
1)	Эта вершина 2. Из неё мы можем пойти в 0 и 3, выберем 0. (2-0)

2)	Из неё два пути: в 1 и в 2, но в 2 мы были, поэтому выбираем 1. (1)

3)	Из 1 опять два пути, один из которых мы прошли. Значит берём 3.(3)

4)	Из вершины 3 есть три пути, но из этих вариантов нам осталось побывать только в 4. (4)

5)	Это последняя вершина графа, но нужно проверить можем ли мы из неё попасть в вершину 2. По матрице видно, что можем, значит Гамильтонов цикл сделан правильно и выглядит так: 2-0-1-3-4-2.
